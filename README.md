# TARAN Boxing Club PWA

Это репозиторий мобильного PWA‑приложения для боксерского клуба «TARAN», разработанного на React, Tailwind CSS и Supabase.

## Основные функции

- **Лента**: новостные посты, поддержка лайков и комментариев, разделение на бесплатный и премиальный контент.
- **Расписание**: просмотр расписания групповых занятий, запись на тренировки, бронирование персональных тренировок.
- **Питание**: платный раздел с расчётом суточной калорийности, генерацией меню, трекером воды и ИИ‑сканером еды.
- **Рейтинги**: ввод результатов нормативов ОФП/СФП, просмотр топ‑10 рейтинга.
- **Профиль**: личные данные, список бронирований, загрузка фото до/после с наложением сетки.
- **PWA**: установка на домашний экран, офлайн‑режим.

## Запуск проекта

1. Склонируйте репозиторий и перейдите в каталог `taran-app`:

   ```bash
   cd taran-app
   ```

2. Установите зависимости (требуется Node.js):

   ```bash
   npm install
   ```

3. Создайте файл `.env` по примеру `.env.example` и укажите параметры Supabase:

   ```env
   VITE_SUPABASE_URL=... # URL проекта
   VITE_SUPABASE_ANON_KEY=... # публичный ключ
   ```

4. Запустите проект в режиме разработки:

   ```bash
   npm run dev
   ```

5. Соберите production‑бандл:

   ```bash
   npm run build
   ```

## Структура

- `src/pages` — страницы приложения (Home, Schedule, Nutrition, Ratings, Profile).
- `src/components` — переиспользуемые компоненты (NavBar, Post и т.д.).
- `src/utils` — вспомогательные файлы (инициализация Supabase).
- `tailwind.config.js` — настройка Tailwind CSS с тёмной темой и неоновыми акцентами.
- `vite.config.js` — конфигурация Vite и подключение плагина PWA.

## Заметки

- Проект является заготовкой: многие функции требуют доработки (AI‑алгоритмы, интеграции с платежами, Apple Health/Google Fit).
- Для корректной работы PWA необходимо развернуть сборку на HTTPS.
- Все интерфейсы локализованы на русский язык.
